Нужно понимать что это лишь информация для тех кто совсем не понимает как работать с CSS редактором. Т.к невозможно научить человека верстке за пару гайдов по документации, для изменения внешнего вида на сайте, этого понимания будет достаточно!

1. Необходимо в свободном месте на сайте нажать на ПКМ и в появившемся меню найти пункт просмотреть пункт
![](https://i.postimg.cc/rsVPF6f7/image.png)
2. В появившемся окне нажимает на клавишу "Select element" (см. скрин)
![](https://i.postimg.cc/tCf28VHC/image.png)
3. Наводимся на нужный нам элемент на сайте и смотрим в окне на то, какой элемент выделился, там показана и сама верстка этого блока и классы этого блока. Далее можем воспользоваться этими данными для использования их в редакторе
![](https://i.postimg.cc/7hJTzhtN/image.png)

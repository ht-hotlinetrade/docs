# Смена заднего фона карточки товара

::: info
Это лишь пример работы с CSS редактором а не готовая реализация
:::

Необходимо указанный ниже код вставить в блок CSS редактора во вкладке **НАСТРОЙКИ->РАСШИРЕННЫЙ РЕДАКТОР** в панели управления магазинов.
Это всего лишь пример изменения фона, вы можете изменять как вам хочется.

**CSS Редактор**

```css{2}
.product-card-image-container {
  background-image: url('https://example.com/image.png') !important;
}
```

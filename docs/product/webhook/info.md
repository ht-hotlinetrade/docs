# Товары > Webhook [Post Request] > Как они работают

### ℹ️ Теоритическая часть
При покупке товара идет отправка POST Request на указанный url с заранее заданными параметрами. Данная система необходима для реализации покупок внешних товаров: для обработки различными ботами, для уведомлений в дискорд вместо покупки и прочее. Данная система поддерживает возможность передач доп параметров, которые автоматически заменяются, например: время покупки, информация о покупателе и прочее. При некорректном ответе (не 200 Success) товар является не выданным, у пользователя покупка не проходит.
